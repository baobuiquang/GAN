# Sorting Algorithms

## Time Complexity
Sorting Algorithms | Worst    | Average  | Best    
------------------ | -------- | -------- | --------
Bubble Sort        | O(n^2)   | Θ(n^2)   | Ω(n)
Insertion Sort     | O(n^2)   | Θ(n^2)   | Ω(n)
Selection Sort     | O(n^2)   | Θ(n^2)   | Ω(n^2)
Merge Sort         | O(nlogn) | Θ(nlogn) | Ω(nlogn)
Quick Sort         | O(n^2)   | Θ(nlogn) | Ω(nlogn)
